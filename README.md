# twinBASIC-Like-Test-Generator
Tool for generating tests for twinBASIC's Like operator

**Usage**

Import all code modules from [src](https://github.com/cristianbuse/twinBASIC-Like-Test-Generator/tree/master/src) and then run method ```Main``` from module [```TestLike.bas```](https://github.com/cristianbuse/twinBASIC-Like-Test-Generator/blob/master/src/TestLike.bas).

Alternatively the [Test Generator for Like operator.xlsm](https://github.com/cristianbuse/twinBASIC-Like-Test-Generator/raw/master/Test%20Generator%20for%20Like%20operator.xlsm) workbook already has the code modules imported.

**Demo Tests**

Demo tests can be found in the [demo tests](https://github.com/cristianbuse/twinBASIC-Like-Test-Generator/tree/master/demo%20tests) folder
